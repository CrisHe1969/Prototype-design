字段

说明

买方企业名称

应收账款金额

该买方企业已上传单据至iDD业务下的单据金额之和；

已提前支付应付账款金额

该买方企业对该供应商“已清算”的“提前回款金额”之和；

提前回款金额 = 可清算金额 - 折扣金额；

AR转化率

AR转化率 = “已清算”单据数/“已报价”单据数；

折扣金额

该买方下针对该供应的“折扣金额”之和；

年利率(%)

该Buyer下，供应商的“已清算”单据的年利率的加权平均数；

年利率的加权平均数 = (R1 * A1 + R2 * A2 + … + Rn * An)/Sum(A1 + A2 + … + An)

R：单笔单据的供应商报价；

A：单笔单据的“可清算金额”；

可清算金额 = 单据金额 * 融资比例(%)

提前回款天数

该Buyer下，供应商的“已清算”单据的提前回款天数的加权平均数；

提前回款天数的加权平均数 =  (T1 * A1 + T2 * A2 + … + Tn * An)/Sum(A1 + A2 + … + An) ;

T：单笔单据的提前支付天数；

A：单笔单据的“可清算金额”；

提前支付天数 = 单据到期日 - 起息日；

字段

说明

总计买方企业

上传该供应商单据至iDD业务的买方企业数量；

提前付款买方企业

已参与提前付款至该供应商的买方企业数量；
