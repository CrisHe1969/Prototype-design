实现年化收益

取“年化收益加权平均数”：

实现年化收益 = (R1 * A1 + R2 * A2 + … + Rn * An)/Sum(A1 + A2 + … + An)

R：单笔单据的年华收益率；

剩余可用资金

剩余可用资金 = 累计已充值金额总额 - 已清算资金总额

应付账款总金额

iDD业务下：Buyer已上传单据金额总额；

提前支付总金额

iDD业务下：Buyer“已清算”单据的提前付款金额之和；

提前付款金额 = 可清算金额  - 折扣收益

                   = 单据金额 * 融资比例 - 折扣金额

总收入

iDD业务下：Buyer实现的折扣收益之和；

折扣收益计算公式：

1.按年化收益计算：折扣收益 = 单据金额 * 年化收益 * (到期日 - 起息日) / 360

2.按折扣率计算：折扣收益 = 单据金额 * 折扣率

平均提前支付天数

取“已清算”单据的“提前支付天数”的加权平均数：

平均提前支付天数 =  (T1 * A1 + T2 * A2 + … + Tn * An)/Sum(A1 + A2 + … + An) ;

T：单笔单据的提前支付天数；

A：单笔单据的“可清算金额”；

提前支付天数 = 单据到期日 - 起息日；

已服务供应商

截止当前时间为止，已提前支付的供应商总和；

资金利用率

iDD业务下：截止当前时间为止该币种 “资金利用率” = 提前支付总金额 / 资金池总额
