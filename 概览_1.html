期望年化收益

取Ops 给买方设置的“期望年化收益”

实现年化收益

取“年化收益加权平均数”：

实现年化收益 = (R1 * A1 + R2 * A2 + … + Rn * An)/Sum(A1 + A2 + … + An)

R：单笔单据的年华收益率；

A：单笔单据的“可清算金额”；

可清算金额 = 单据金额 * 融资比例(%)

资金池总额

取当前币种下Buyer 累计已充值金额总额；

剩余可用资金

取当前币种下剩余可用资金；

剩余可用资金 = 累计已充值金额总额 - 已清算资金总额

应付账款总金额

iDD业务下：Buyer已上传单据金额总额；

提前支付总金额

iDD业务下：Buyer已清算金额之和；

总收入

iDD业务下：Buyer实现的折扣收益之和；

折扣收益计算公式：

1.按年化收益计算：折扣收益 = 单据金额 * 年化收益 * (到期日 - 起息日) / 360

2.按折扣率计算：折扣收益 = 单据金额 * 折扣率

资金利用率

iDD业务下：截止当前时间为止该币种 “资金利用率” = 提前支付总金额 / 资金池总额

平均提前支付天数

取“提前支付天数”的加权平均数：

平均提前支付天数 =  (T1 * A1 + T2 * A2 + … + Tn * An)/Sum(A1 + A2 + … + An) ;

T：单笔单据的提前支付天数；

A：单笔单据的“可清算金额”；

提前支付天数 = 单据到期日 - 起息日；
