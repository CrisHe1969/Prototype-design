计算规则

年化利率

截止当前系统时间为止：选中币种下，该供应商已清算单据的“加权平均年化利率”；

年利率的加权平均数 = (R1 * A1 + R2 * A2 + … + Rn * An)/Sum(A1 + A2 + … + An)

R：单笔单据的供应商报价；

A：单笔单据的“可清算金额”；

可清算金额 = 单据金额 * 融资比例(%)

应收账款总金额

截止当前系统时间为止：该供应商iDD业务下累计上传单据金额之和；

(若包含不同币种，则统一按照中行实时汇率换算成美元进行计算)

提前贴现总金额

截止当前系统时间为止：选中币种下，该供应商已清算单据的“提前回款金额”之和；

提前回款金额 = 可清算金额 - 折扣金额；

可清算金额 = 单据金额 * 融资比例(%)

折扣金额

截止当前系统时间为止：选中币种下，该供应商已清算单据的“折扣金额”之和；

1. 报价方式为[年化利率(%)]：折扣金额 = 可清算金额 * 年化利率(%) * (单据到期日-起息日) / 360

2. 报价方式为[折扣率(%)]：折扣金额 = 可清算金额 * 折扣率(%)

单笔AR平均提前回款天数

截止当前系统时间为止：选中币种下，该供应商已清算单据的“AR提前回款天数”的加权平均数；

提前回款天数的加权平均数 =  (T1 * A1 + T2 * A2 + … + Tn * An)/Sum(A1 + A2 + … + An) ;

T：单笔单据的提前支付天数；

A：单笔单据的“可清算金额”；

提前支付天数 = 单据到期日 - 起息日；
